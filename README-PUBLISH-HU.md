# Nova Fordító publikálás

Az oldal statikus weboldal, ezért bármelyik egyszerű statikus hostingra feltölthető.

## Ajánlott

### Netlify
1. Nyisd meg a Netlify oldalt.
2. Hozz létre egy új oldalt drag and drop feltöltéssel.
3. Töltsd fel ezt a projektmappát vagy a kész ZIP csomagot.
4. A publikálás után kapsz egy publikus linket.

### Vercel
1. Nyisd meg a Vercel oldalt.
2. Importáld ezt a mappát vagy a feltöltött projektet.
3. A publikálás után kapsz egy publikus linket.

## Fontos

- Attól, hogy publikálva van, a Google kereső nem azonnal fogja megjeleníteni.
- A keresőben való megjelenéshez publikus URL kell, és idő, amíg a Google feltérképezi az oldalt.
- Ha saját domaint használsz, később érdemes majd `sitemap.xml` fájlt is hozzáadni a végleges domainnel.
